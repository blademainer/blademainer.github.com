### v0.2.8

- Update node modules: moment to 2.10.6.

### v0.2.7

- Add `--lteb [pct]` support to query symbols that make 100*(s.price - s.cheap)/s.price <= pct.

- Update node modules:blessed,blessed-contrib,moment,request.
